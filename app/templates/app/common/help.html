网站操作说明

  1. 系统软硬件配置

    系统运行软硬件配置要求如下表1：系统软硬件配置表所示
    操作系统 Linux2.6+ 或 windows 7+
    数据库系统 MySQL5.5+
    Python 3.5+
    Mysqlclient 1.3.10+
    Django 1.10+
    Pillow 4.0.0+
    内存 4G及以上
    硬盘 500G及以上

  2. 操作说明

    系统的运行前需要先配置好系统运行依赖环境，包括安装配置数据库，安装Python语言并安装依赖库，包括后台开发的Django框架，数据库操作依赖的Mysqlclient库，验证码生成依赖的Pillow库。此外，初始运行系统时，需要进行系统的初始化操作，需要执行Django框架的数据模型初始化命令。

  3. 数据库安装配置

    数据库安装步骤：

    1. 添加mysql组和mysql用户
    2. 解压编译安装mysql
    3. 修改mysql文件夹权限
    4. 执行mysql_install_db脚本进行安装
    5. 设置service启动
    6. mysql路径写入环境变量PATH

    数据库安装执行脚本如下：

    1. > groupadd mysql
      useradd -r -g mysql -s /bin/false mysql
    2. > tar zxvf mysql-VERSION.tar.gz
      cd mysql-VERSION
      cmake (../mysql-src -LAH)
      cmake (../mysql-src -LAH)
    3. > cd /usr/local/mysql
      chown -R mysql .
      chgrp -R mysql .
    4. > scripts/mysql_install_db --user=mysql --basedir=/usr/local/mysql
      --datadir=/usr/local/mysql/data
      chown -R root .
      chown -R mysql data
      bin/mysqld_safe --user=mysql &
    5. > cp support-files/mysql.server /etc/init.d/mysql.server
    6. > vim ~/.bashrc 增加 PATH=$PATH:mysql_install_dir/bin/

    数据库安装配置好后登录数据库建立新的数据库eshop。系统初次运行时需要往数据库admin表插入新超级管理员记录，管理员密码需要进行sha512加密。

  4. 系统运行环境配置

    系统运行环境需要安装有Python，然后安装pip，接着安装各个依赖库包括Django,Mysqlclient,Pillow库等系统运行所需依赖库。

    • 安装命令如下所示：
    • 安装Django：
    • > pip install Django
    • 安装Mysqlclient：
    • > pip install mysqlclient
    • 安装Pillow：
    • > pip install pillow

    安装完系统后需要初始化系统，需要在项目根目录下执行以下Django框架数据模型初始化命令：

    1. > manage.py makemigrations
    2. > manage.py migrate
